Welcome

Hi and welcome to the official Apricitine style guide! This guide will teach my standards for code formatting, keyword usage, general standards, etc.
The langauges in this guide will be the base languages, no frameworks like Vue, React, or Sass as an example.

JavaScript

Let's begin with variables.


All global variables should be initialized and valuated at the beginning of the code using const, var if neccessary. All set variables should have their own keyword, only use commas for variables with similar values. If using a DOM method like getElementById() or querySelectorAll(), that means that the return value will be an HTMLCollection or HTMLElement object. Always put underscores after these variable names to distinguish that these variables are defined for DOM traversal. Try to keep all data types together and add blank lines after each data type. If there is only being one variable assigned to a data type, it can go with the others.



Next, function/method styling.

Normal callable global functions should always be defined like this: function name(parameter1, parameter2) {} When using IIFEs, always use parantheses to encapsulate the anonymous function. Also, make sure that there is a space between the function keyword and parameter parantheses: (function () {})(); .
